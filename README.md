# Messenger

## Project setup
Для запуска пректа необходимо выполнить команды:

```
yarn install
yarn run serve
```

После проект будет доступен по хосту: http://localhost:8080/



### Compiles and minifies for production
```
yarn run build
```

### Run your tests
```
yarn run test
```

### Lints and fixes files
```
yarn run lint
```
